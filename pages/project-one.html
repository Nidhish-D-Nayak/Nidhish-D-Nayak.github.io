Portfolio

About the project

The reason I chose this as my first project was because it is essential to have a portfolio website. It allows me to provide information about myself and showcase my best work with relevant skills and experience.

I also wanted to get some hands on experience with responsive design and up my skills. Hence, this was the perfect opportunity to grab!

Issues faced & fixed

I faced multiple issues with responsiveness as it got more and more complex. I used grid and flexbox to resolve these problems in skills and projects section. For the other parts I have used media queries for various sized devices.

Since I wanted to keep the javascript as simple as possible. To make the portfolio a little more interesting, I decided to use a javascript library called AnimeJS that allows you to easily implement light weight animations.
